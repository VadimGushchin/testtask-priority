# testtask-priority

## Задача

Предсказание вероятности целевого действия в течение 5 дней после назначения обращения.  
**Метрика:** Daily Average Precision (усреднение AP по дням).  
**Ограничение:** запрещены признаки, недоступные на момент скоринга (future leakage).

## Финальное решение (`solution.ipynb`)

Пайплайн включает:

1. **Инженерия признаков:**  
   - Отношения (views per contact, favorites per view, search per view) для окон 7d, 14d, 30d, 90d.  
   - Разности между окнами.  
   - Циклические признаки (sin/cos для часа и дня недели).  
   - Скользящие средние конверсии по категориям (регион, источник, канал) за 7, 14 и 30 дней.

2. **Признаки из событий:**  
   - Агрегации по окнам (15m, 1h, 3h, 6h, 24h) – общее число событий и по типам.  
   - Время с последнего события каждого типа.  
   - Разнообразие типов за 24 часа и бинарные флаги за 2 часа.  
   - Статистики по `src_slot` (mean, std, min, max) и число уникальных `ctx_seq`.

3. **Отбор признаков:**  
   - По важности `gain` LightGBM (оставлено 95% кумулятивной важности) – размерность сокращена с 194 до 99 признаков.

4. **Оптимизация гиперпараметров:**  
   - Optuna с TimeSeriesSplit (5 фолдов) и штрафом за переобучение (`alpha = 0.35`).  
   - Найдены оптимальные параметры для LightGBM, XGBoost, CatBoost и LogisticRegression.

5. **Ансамблирование:**  
   - Равные веса (усреднение) – даёт баланс качества и переобучения.  
   - Финальное обучение на объединённых train+val с лучшими параметрами.

## Результаты

На валидационной выборке (2026-04-20 – 2026-04-22):

| Модель              | Val AP  | Overfit (Train - Val) |
|---------------------|---------|------------------------|
| LightGBM (tuned)    | 0.6443  | 0.0917                 |
| XGBoost (tuned)     | 0.6360  | 0.0543                 |
| CatBoost (tuned)    | 0.5945  | 0.0302                 |
| LogisticRegression  | 0.5746  | -0.0149                |
| **Ансамбль (равные)** | **0.6435** | **0.0433**             |

Ансамбль показывает лучший баланс между качеством и устойчивостью к переобучению.
